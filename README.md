# GitHub Repository Collaboration Project

## Project Description
This project is created as part of the Software Engineering assignment to demonstrate the use of Git and GitHub for version control and team collaboration. The project shows how multiple team members can work together using branches, commits, pull requests, and merging in a shared repository.

## Project Goal
The goal of this project is to practice collaborative software development using Git and GitHub, including repository management, branching, committing changes, creating pull requests, and merging code into the main branch.
